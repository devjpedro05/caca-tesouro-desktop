# Caça ao Tesouro em Redes de Túneis

Um jogo de tabuleiro educacional baseado em grafos, desenvolvido em Python e PySide6.

## Estrutura do Projeto

- `core/`: Lógica do jogo (grafo, algoritmos, estado do jogo).
- `ui/`: Interface gráfica (PySide6).
- `assets/`: Ícones e fontes.
- `tests/`: Testes unitários.

## Como Rodar

1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

2. Execute o jogo:
   ```bash
   python -m ui.main_qt
   ```
